Modern Technológiák

Ez a kód egy egyszerű React komponenst mutat be, amely egy számlálót tartalmaz. A komponens a useState hookot használja a számláló állapotának kezelésére. Amikor a gombra kattintanak, a számláló értéke eggyel növekszik. A React egy népszerű JavaScript könyvtár, amely lehetővé teszi a felhasználói felületek komponens alapú fejlesztését.

// React Hooks példa
<script type="text/babel">
import React, { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
    return (
        <div>
            <button onClick={() => setCount(count + 1)}>Click me</button>
            {count}
        </div>
    );
}

ReactDOM.render(<Counter />, document.getElementById('react-root'));
</script>

Aszinkron JavaScript

Ez a kód egy aszinkron függvényt mutat be, amely adatokat kér le egy API-ból. A függvény a fetch metódust használja az adatok lekérésére, és a try...catch blokkot a hibakezelésre. Az aszinkron JavaScript lehetővé teszi a nem blokkoló műveletek végrehajtását, ami javítja az alkalmazás teljesítményét és felhasználói élményét.

// Aszinkron JavaScript példa
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Error fetching data:', error);
    }
}

fetchData();

DOM Manipuláció

Ez a kód egy egyszerű DOM manipulációs példát mutat be. Amikor a gombra kattintanak, egy figyelmeztető üzenet jelenik meg. A DOM (Document Object Model) lehetővé teszi a HTML dokumentumok programozott módosítását és interakcióját a JavaScript segítségével.

// DOM Manipuláció példa
document.addEventListener('DOMContentLoaded', () => {
    const button = document.querySelector('button');
    button.addEventListener('click', () => {
        alert('Button clicked!');
    });
});

Promises

Ez a kód bemutatja a JavaScript Promises használatát. A Promise egy objektum, amely egy aszinkron művelet végrehajtását képviseli, és lehetővé teszi a művelet sikeres vagy sikertelen befejezésének kezelését.

// Promise példa
const myPromise = new Promise((resolve, reject) => {
    const success = true; // Állítsd be true-ra vagy false-ra a siker vagy hiba szimulálásához
    if (success) {
        resolve('A művelet sikeres volt!');
    } else {
        reject('A művelet sikertelen volt.');
    }
});

myPromise
    .then(result => {
        console.log(result);
    })
    .catch(error => {
        console.error(error);
    });

Async/Await

Ez a kód bemutatja az async/await szintaxist, amely egyszerűsíti az aszinkron kód írását. Az async kulcsszó egy függvényt aszinkronná tesz, az await kulcsszó pedig megvárja egy Promise teljesülését.

// Async/Await példa
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Error fetching data:', error);
    }
}

fetchData();
